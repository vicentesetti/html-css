Usando links externos

Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal gratuito no YouTube

Envelopando o texto que quero linkar com <a>, preciso inserir o link que quero usar

Também posso escolher se o link será aberto na janela atual, ou em uma nova janela, usando target="_blank" rel="external" após o link dentro de <a>

target="_blank": indica que o link será aberto em uma nova aba

rel="external": indica que o link é externo
