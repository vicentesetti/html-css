Principais Formatações

Negrito / destaque

Nesta frase temos um termo em negruito usando a tag <B> (não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande , com a tag obsoleta e não semântica <big> e um texto pequeno com a tag semântica e não obsoleta<small> nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, com a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag <ins>.

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O, usando a tag <sub>.
