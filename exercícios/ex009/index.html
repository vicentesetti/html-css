  8. Almoçar;
  9. Estudar;

Usando a tag <ol> para criar a lista ordenada, e usando a tag <li> para colocar os itens da lista.

Posso usar a tag type="1/A/a/I/i" dentro da <ol> para selecionar o tipo de numeração da lista.

Usando a tag <ul> para criar a lista ordenada, e usando a tag <li> para colocar os itens da lista.

Posso usar a tag type="disc/circle/square" dentro da <ul> para selecionar o tipo de marcação da lista.
